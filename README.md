# Heart Disease Prediction using Machine Learning

## Internship

Growfinix Machine Learning Internship

## Task

Task 1: Healthcare – Heart Disease Prediction (Classification)

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn

## Machine Learning Models

* Logistic Regression
* Support Vector Machine (SVM)

## Description

This project predicts the risk of heart disease using patient health information such as Age, Blood Pressure, and Cholesterol. The project demonstrates data creation, preprocessing, feature scaling, model training, and prediction.

## Steps Performed

1. Import required libraries.
2. Create a sample dataset.
3. Check for missing values.
4. Separate features and target.
5. Split the dataset into training and testing sets.
6. Apply feature scaling.
7. Train Logistic Regression model.
8. Train SVM model.
9. Compare the prediction accuracy.

## Output

The program displays:

* Sample dataset
* Missing value check
* Logistic Regression Accuracy
* SVM Accuracy
* Task Completed Successfully
